找不到org.apache.http

原因：

android 6.0 (sdk版本 23+)已经不支持httpclient这个类了
搜索关键词：sdk 23 httpclient，可以看到相关的信息

解决方法：

你可以使用URLConnection，或者 用sdk 22

(compile'com.android.support:appcompat-v7:22.2.0')

如果必须使用sdk 23,添加如下代码：

android {
    ....
    useLibrary 'org.apache.http.legacy'
}
